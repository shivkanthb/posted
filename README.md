### Getting Set Up

1. Fork this repository on Github.
2. Clone repo `git clone https://github.com/YOURUSERNAME/posted.git`
3. `cd posted`
4. Initialize and build :
```
npm install
npm run build
npm start
```

Then go to http://localhost:3000/ to view your posts.
